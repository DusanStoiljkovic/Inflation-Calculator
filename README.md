# Inflation Calculator

A simple web-based inflation calculator that estimates the future value of money based on an annual inflation rate.

## Features
- Input validation
- Percentage-based inflation calculation
- Clean UI
- Responsive layout

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
